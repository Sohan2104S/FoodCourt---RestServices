System Overview & Request Processing Flow

🛠️ Tech Stack

The backend is built with modern, robust technologies ensuring scalability and maintainability.

Java 17+ Spring Boot 3 RESTful APIs In-Memory DataStore Role-based Access Control

🧩 Core Components

The system is divided into clean architectural layers for the Restaurant Application:

  • Controllers: Order, Restaurant, Payment - handle incoming HTTP requests.
  • Services: AccessControl, PaymentGateway, Email - contain the business logic.
  • DataStore: Centralized in-memory repository simulating database state.
